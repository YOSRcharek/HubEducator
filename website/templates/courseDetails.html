Multimodal Virtual Assistant

An intelligent assistant to help students via text, voice, and image.

  • Number of hours: 25h
  • Price: €120

Course Description

This course allows students to explore and use a multimodal virtual assistant capable of understanding questions via text, voice, and images. It also provides automated exercises, personalized tracking, and certificate generation at the end of the program.

Course Features

1. Text Interaction

Students can ask questions via an integrated chat area.

2. Voice Interaction

Voice recognition to ask a question directly to the assistant.

3. Image Interaction

Ability to upload an image (e.g., written exercise).

Assistant Responses
  • Explanatory text
  • Generated diagram or image
  • Audio (text-to-speech)
Course Management

Chapters/Sections, associated documents (PDFs, videos, images). Access according to level or enrollment.

Exercise Generation

MCQs, True/False, Fill-in-the-Blank, Open-ended problems. Each exercise includes the question, possible answers, and correction.

Assessments & Personalized Tracking

Teachers create quizzes/exams. Students take assessments with automatic correction and feedback. Dashboard to track exercises, results, strengths, and weaknesses.
